# LUXURY CLOTHING

### ✔ 2.서비스소개 
OCR 기반 영수증 활용 중고 명품 판매 플랫폼
위탁 서비스를 통한 편리한 거래 기능
- 다양한 상품을 개인에게 맞는 이번 주 상품 추천 구매 기능
- 적정한 가격과 신뢰를 기반한 명품 의류 판매
- 검색 및 필터링 기능  

### ✔ 3.프로젝트기간
2024 1. 16 ~ 2024. 2. 26

### ✔ 4.주요기능
- OCR을 이용한 영수증 인증
- 위탁 서비스
- 매입 서비스
